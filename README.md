# Cream Social Media APP
Flutter 적응기를 위한 소셜 미디어 서비스 앱입니다.

## 라이브러리
- image_picker : 갤러리로부터 이미지를 받아, 업로드를 위해 사용됩니다. ( 카메라, 프로필, 포스트 등 )
- go_router : 다양한 화면을 한 곳에 정의하여 복잡성을 줄이고, 라우팅을 위해 사용됩니다.
- intl : 날짜 변환을 위해 사용됩니다.
- timeago : 등록된 날짜의 정수 값을 기준으로 경과한 시간을 나타내기 위해 사용됩니다.
- dio : 데이터 패칭을 위해 사용됩니다.
- dio_cookie_manager : 쿠키 데이터를 관리하고 세션 관리를 위해 함께 사용됩니다.
- cookie_jar : dio 관련 패키지와 쉽게 통합되어 인터셉터 쿠키 관리 및 응답 간의 쿠키 처리르 위해 사용됩니다.
- path_provider : 쿠키 정보를 찾고 앱 데이터 관련 디렉토리를 찾기 위해 사용됩니다.
